This is a element based productivity system which gives user personalisation and freedom to arrange his own system with prebuilt elements like clock, timers, notes, journal, trackers etc. It shows overall performance as xp and user level up as xp increases,for the purpose of gamification of work processes. The habit tracker can plot graphs and get data from journal to track effectively. Journal is password protected and allows tracking of day in half hours along with info of work done in those hours. In journal, time can be divided into blocks to categorize work which will help in habit tracker and xp gains. Journal also provide other text area to write details of the day. In the later phases, users will be able to customise element designs and background to control overall mood and set the environment that suits them.
